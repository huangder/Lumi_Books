
# UI 设计与开发规范：Android 阅读应用（Apple Books 风格）

## 角色与设计哲学

你是一位精通极简主义、高质感出版物级移动端界面的资深 UI/UX 设计师兼前端工程师。你的任务是为一款高端 Android 阅读 App 设计/实现三个核心页面。该应用需要完美复现 Apple Books 优雅的“杂志感”与大留白美学，同时深度适配 Android 生态（采用 Material 3 流体物理动效与类似 iOS 的细腻微交互）。

---

## 第一部分：全局视觉 Token（应用于所有页面）

* **色彩体系 (Color Palette)：**
* `Window Background`（全局背景）：`#FBFBFC`（极微弱的暖白，有效缓解长时间阅读的视觉疲劳）。
* `Card Background`（卡片背景）：`#FFFFFF`（纯白，通过微弱的物理阴影在暖白背景上拉开层级）。
* `Text Primary`（主要文字）：`#000000`（高对比度，确保极致的纯净与易读性）。
* `Text Secondary`（次要文字）：`#6E6E73`（经典高级灰，用于副标题、阅读进度、元数据及辅助文本）。
* `Accent Color`（强调色）：`#007AFF`（电子标准蓝，用于进度条、激活状态和焦点元素）。
* `Divider/Track`（分割线/轨道）：`#E5E5EA`（浅灰，用于隐蔽的边界线与未填充的进度条轨道）。


* **字体排版 (Typography)：**
* **UI/系统文本：** 干净、现代的无衬线体（如 Roboto、Inter 或 MiSans）。
* **书名与核心大标题：** 优雅、古典的衬线体（如思源宋体 / Source Han Serif），营造纸质书印刷的质感。


* **高度与阴影 (Elevation & Shadows)：** 严格杜绝 Android 原生生硬的投影。必须使用极度细腻、扩散范围大（Blur 较大）且透明度极低的阴影：等效于 `Elevation: 2dp`，`Blur: 12dp`，`Opacity: 3%~5%`。
* **全局圆角规范 (Corner Radius)：** 大卡片 = `16dp` 至 `20dp`；书籍封面 = `6dp` 至 `8dp`。

## 第二部分：核心页面详细需求

### 页面 1：主页 (Home Screen)

* **设计目标：** 极高的留白率，让用户能瞬间专注于当前阅读的书籍，并提供每日阅读目标的激励。
* **布局结构（自上而下）：**
1. **页头区域 (Header)：**
* 左侧为巨大的衬线体标题“主页” (`32sp`, Bold, `Letter-spacing: -0.02em`)。
* 右侧对齐：一个极简的环形阅读目标进度条（圆环中心显示当前连续打卡天数或等级），紧邻精致的用户头像。


2. **系统通知/导入提示（可关闭）：**
* 一个浅灰色调的微悬浮容器（`#F2F2F7`, `CornerRadius: 12dp`），内置一个小书本图标与辅助提示语（例如：“在此处管理您的本地 PDF 和 EPUB 图书”）。


3. **“继续阅读” (Continue Reading) 模块：**
* 采用单个、突出的横向大卡片承载。
* 左侧：书籍封面（严格执行 **3:4 比例**，封面右侧边缘带有 1px 的垂直暗影以模拟书脊厚度）。
* 右侧：书名（衬线体，`16sp`，Semi-Bold）、作者（无衬线体，`#6E6E73`，`14sp`）以及一行清晰的灰色进度文本（如“图书 · 26%”）。最右侧放置一个精致的三点菜单按钮。


4. **“之前读过” (Read Before) 模块：**
* 横向水平滚动列表（RecyclerView / LazyRow）。
* 紧凑的纯白卡片，内部包含左侧稍小的书籍封面，右侧紧凑排列书名、作者与已读百分比，卡片间保持舒适的间距。


5. **“阅读目标” (Reading Goal) 卡片：**
* 一个留白极宽敞的独立容器。
* 中心：一个完美的、呈拱门状的半圆形弧线进度条（使用强调蓝）。
* 弧线内部：巨大的加粗计时文本“2:41” (`36sp`，无衬线体)，下方紧跟灰色小字阶单位与跳转箭头“(目标 5分钟) >”。
* 弧线下方：一个纯黑色的胶囊形按钮（背景 `#000000`，文字 `#FFFFFF`，`CornerRadius: 24dp`），内写“继续阅读 \n [具体书名]”。
* 卡片最底部：一排水平居中的星期缩写小圆圈 `(周日, 周一, ... 周六)`，已打卡天数用蓝色线条外圈高亮，用于追踪本周打卡状态。


6. **“今年读过的图书” (Books Read This Year) 网格：**
* 小标题“今年读过的图书”，左侧带有一个轻量化的返回箭头。
* 下方为一个 3 列的网格占位区域，显示干净的灰色虚线/实体框，正中心带有优雅的大号数字字体“1”、“2”、“3”，用以承载今年读完的图书排行。





### 页面 2：阅读统计页 (Reading Statistics Screen)

* **设计目标：** 将复杂的图表与阅读数据转化为极具杂志排版感的现代画册布局，杜绝杂乱。
* **布局结构（自上而下）：**
1. **页头区域 (Header)：**
* 左上角一个轻量的返回键 (`<`)。
* 大号加粗衬线体标题“阅读时长与书籍统计” (`28sp`, `Letter-spacing: -0.01em`)。


2. **“本周阅读概览” (Weekly Overview) 模块：**
* 左右分割的现代化横向布局。
* **左侧：** 巨大的核心数据块。“12 小时 \n 45 分钟”，使用超粗无衬线体 (`36sp`)。数据下方紧跟一个精致的同比增长指示器（如绿色的 “↑ 21%”）。
* **右侧：** 一个极简的周阅读柱状图。包含 7 根垂直柱体（周一至周日）。*硬性视觉要求：* 每根柱子的顶端必须为顺滑的圆角（`Radius: 4dp`）。去除所有多余的网格线、Y轴刻度和背景框，仅在图表底部保留极干净的星期标签 `(周一, 三, 四, 五, 六, 七)`。


3. **“最常阅读的书籍” (Most Read Books) 卡片：**
* 一块纯白的大卡片，内部包含垂直排列的常读书籍列表。
* 每行包含一个微型的 3:4 书籍封面、书名、作者，以及具体的统计指标（如“已读: 48%”或“平均每日: 45分钟”）。行与行之间由极淡的 0.5dp 分割线隔开。


4. **“完成进度” (Completion Progress) 卡片：**
* 包含两个优雅的线性进度条模块：
* 模块 1：“本年目标 (15/20本)”。右侧配以一个小巧的书堆 Emoji/图标。下方是一根极细的水平进度条（强调蓝轨道配合 `#E5E5EA` 未填充背景）。
* 模块 2：“本周目标 (10/12小时)”。下方配以同样高质感的线性进度条。







### 页面 3：书架页 (Bookshelf/Library Screen)

* **设计目标：** 打造一个极具触觉感知、干净的数字书房，将“书籍封面”本身作为绝对的视觉主角。
* **布局结构（自上而下）：**
1. **页头区域 (Header)：**
* 大号衬线体标题“书库” (`32sp`, Bold)。
* 右侧对齐的功能按钮：一个列表/网格视图切换图标，以及一个“排序/筛选”的文本按钮（如“收藏夹”或“最近”）。


2. **顶部导航标签 (Sub-headers)：**
* 一个极简的水平横向分段文本行：“全部图书”、“下载内容”、“PDF”、“收藏”。当前选中的标签加粗并呈纯黑色，未选中的标签为灰色 `#6E6E73`，彼此留有宽敞的横向间距。


3. **核心书架网格 (Main Book Grid)：**
* 一个流式的 3 列标准网格布局，拥有极其慷慨的纵向和横向四周留白（Padding）。
* 每个书籍单元格*仅由以下元素*构成：
* 高分辨率的书籍封面（严格 3:4 比例）。
* 封面后方带有极自然的、向外扩散的微弱环境光阴影。
* 封面正下方：书名（无衬线体，`14sp`，Semi-Bold，最多展示两行，超出省略）以及作者名（`12sp`，`#6E6E73`，严格单行）。
* 在封面艺术图的最底部边缘，可以放置一根极其纤细的、近乎隐形的进度指示线（或在右下角标明轻量的 “已读 12%”）。




4. **添加/空白状态：**
* 在网格的最末尾，提供一个虚线框或浅灰色的极简卡片，中心仅带有一个轻量化的 `+` 图标，用于导入本地书籍。

## 第三部分：浮动胶囊导航栏规范 (Floating Tab Bar)

* **形态与几何 (Geometry)：** 彻底摒弃 Android 传统通栏式的底部导航栏，采用 **微悬浮胶囊 / 分段控制条 (Floating Capsule)** 样式。
* `宽度`：非全宽。卡片左右边缘需与屏幕边缘保持明显的间距（屏幕两侧留出 `Margin: 24dp`）。
* `高度`：固定在 `56dp` 至 `64dp` 之间。
* `圆角`：`28dp` 至 `32dp`（两侧形成完美的半圆弧胶囊）。


* **材质特性 (Material)：** 采用高级半透明纯白（`#FFFFFF` 配合 90% 的不透明度 Alpha 属性），同时在 Android 12+ 上必须开启实时高斯模糊（`RenderEffect.createBlurEffect`）。当用户滑动上方的书籍列表时，书籍可以从导航栏背后隐约划过，营造极佳的空气感。
* **标签与交互状态 (Tabs & Interaction)：**
* 常驻三个 Tab：“主页”、“书库”、“阅读统计”。
* **未选中状态：** 图标为细单线（Linear Outline），文字颜色为 `#6E6E73`，字重为 Regular。
* **选中状态：** 图标转变为面性填充（Solid）或在图标外围包裹一层**极淡的横向圆形胶囊高亮底色背景**（如 image.png 所示）。文字变为纯黑 `#000000`，字重加粗。

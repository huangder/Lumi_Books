# 2026-07-21 液态玻璃应用主题（第一阶段）

## 更新内容

- 新增 `liquid_glass` 应用外观主题和 `liquid_glass_transparency` DataStore 偏好，默认通透度为 55%。主题和通透度会实时写入设置状态，应用重启后保持。
- 在“显示与外观”的主题选择下方增加“玻璃通透度”卡片。仅当选中液态玻璃主题时显示；卡片以展开、上滑和淡入同步进入，后续设置卡片随布局高度平滑下移。
- 将液态玻璃主题的底部 Tab 栏改为 72dp 图文胶囊：图标在上、标题在下，选中项使用会横向滑动的中性玻璃透镜，不再使用蓝色填充。通透度同时控制表面不透明度和背景模糊强度。
- 接入 `io.github.kyant0:backdrop:1.0.6` 官方实现。页面内容由 `layerBackdrop` 单独记录，底栏依次执行 `vibrancy -> blur -> lens`，选中透镜再组合页面与底栏的背景；图标和文字在效果层之后绘制，避免折射前景导致重影和重叠。
- 删除临时自制 AGSL 后处理方案。当前外层胶囊和选中胶囊都使用 Backdrop 的真实背景采样与边缘折射，透明度继续映射到玻璃表面不透明度和模糊半径。
- 为满足官方 Backdrop 版本要求并兼容本机 Java 24，工具链升级为 Gradle 8.14.3、AGP 8.13.2、Kotlin/Compose Compiler/KSP 2.3.10、Compose BOM 2026.06.01、compileSdk 36。Hilt 同步升级到 2.58，Room 同步升级到 2.8.4；targetSdk 仍为 35，minSdk 仍为 26，源码字节码目标仍为 JVM 17。
- 修复英文、日文、韩文资源中的重复 `book_author_unknown` 定义。该重复项在新版资源合并器下会直接阻断构建，保留原有较早译文，不改变业务行为。
- 阅读器快捷菜单、底部面板和可拖拽液态开关尚未接入，将在后续阶段按统一浮层状态机实现。

## 底部 Tab 拖动交互修正

- 对照 AndroidLiquidGlass 仓库的 `LiquidBottomTabs` 与 `DampedDragAnimation` 示例，移植同类阻尼拖动状态：按下立即产生 `pressProgress`，透镜弹性放大；拖动时同时记录位置与速度并产生方向形变；松手从当前连续位置弹簧吸附到最近 Tab，不再先跳回旧索引。
- 将液态底栏改为三个独立兄弟图层：外层玻璃、正常黑色图文、由 `layerBackdrop` 记录的蓝色图文。选中透镜通过 `rememberCombinedBackdrop` 采样蓝色图文层，因此经过的图标和文字会在透镜范围内变蓝并参与棱镜折射。
- 外层玻璃不再作为透镜的父效果层，避免 `drawBackdrop` 的胶囊形状裁剪放大后的透镜。透镜按官方示例比例放大时可以越过底栏上下边界。
- 液态主题图标调整为圆角 `Home`、`AutoStories`、`Leaderboard`，图标增大到 28dp，标签调整为 11sp 中等字重；浅色选中态使用 10% 中性黑表面，提升白色页面上的辨识度。
- `compileDebugKotlin` 与 `assembleDebug` 均通过，已覆盖安装到 Android API 36 真机。静止状态截图为 `app/build/liquid-glass-tab-unclipped.png`；真机禁止 ADB 注入触摸，按住和拖动过程由用户直接在设备上验证。
- 后续真机反馈修正：外层玻璃改为独立兄弟层，放大透镜不再被外轮廓裁剪；图文整体上移 4dp，使图标与标签的组合视觉中心对齐胶囊中心。
- 取消底栏额外 8dp 横向内容内边距，三等分基于完整胶囊宽度计算；选中透镜统一使用四周 3dp 内边距，首尾选中态与外轮廓的上、下、左/右留白一致。
- 将拖动跟手弹簧与点击/松手吸附弹簧拆分：跟手维持高刚度，吸附降低到 `dampingRatio=0.80`、`stiffness=240`；透镜进入目标约 5% 范围时即开始缩小，使横向移动与回弹收缩自然重叠，避免到位后停顿。
- 最终静止状态截图更新为 `app/build/liquid-glass-tab-spacing-fixed.png`，并再次覆盖安装到真机。
- 液态底栏选中内容由固定蓝色改为 Lumi 粉色。浅色模式的 Backdrop 采样层使用补偿色 `#FF6868`，抵消中性灰表面的约 10% 压暗后，真机视觉接近 Lumi 原始强调色 `#E85D5D`；深色模式沿用 `AppColors.Accent`。验证截图为 `app/build/liquid-glass-tab-lumi-pink.png`。

## 阅读器液态玻璃菜单

- 新增通用 `LiquidGlassSurface`，以 Backdrop 的 `vibrancy -> blur -> lens` 处理阅读页背景，并提供高光、内阴影、外阴影及按压弹性缩放；通透度同时控制玻璃表面色和模糊半径。
- EPUB/TXT 与 PDF 阅读器分别记录正文背景，将顶部返回、听书、翻页方向、书签按钮以及底部目录、搜索、笔记、主题、PDF 转换等胶囊切换为液态玻璃表面。
- 修复 `ReaderRouter` 内层主题未透传的问题。此前内层 `EBookReaderTheme` 将 `LocalAppTheme` 重置为默认 `lumi`，导致阅读器实际走普通灰色胶囊与渐变遮罩；现在完整继承应用主题和玻璃通透度。
- 液态玻璃主题不再绘制阅读器顶部、底部整片渐变遮罩，胶囊直接采样原始书页；普通主题维持原有阅读辅助渐变。
- PDF 目录进度由错误的 `currentPage / pageCount` 修正为 `(currentPage + 1) / pageCount`。液态主题下进度改为胶囊底部细轨，避免整块粉色填充遮挡玻璃和文字；普通主题维持原填充样式。
- 打开目录、笔记、搜索、主题或 PDF 转换面板时，同一事件中触发旧菜单退出与新面板进入；主题面板进入高级设置时也同步退出，避免透明面板叠加显示。
- 静止态折射参数对齐官方 `LiquidButton` 示例的 12dp/24dp，色散只在按压时开启，避免常态出现过重彩边。
- 为阅读器液态胶囊增加内容衬底渐变，绘制顺序为“书页折射 -> 半透明衬底 -> 图标/文字”。PDF 使用当前应用深浅模式对应的白色或黑色，EPUB/TXT 使用实际阅读背景色或自定义背景主色；普通阅读胶囊衬底保持约 18% 基础不透明度，上沿略亮、下沿略透明。
- 增强阅读胶囊外阴影，并将轮廓改为上亮下暗的高光渐变描边。主页液态 Tab 外轮廓同步增强上沿高光和 Backdrop 静态高光，不改变原有尺寸、拖动与吸附参数。
- TTS 播放面板接入阅读页 Backdrop，液态主题使用约 85% 的阅读背景色渐变衬底与玻璃高光/阴影；其他主题继续使用原有实体背景和阴影。
- 液态主题三个主页面底部均使用左侧 72dp 高的三项 Tab 胶囊与右侧 72dp 圆形导入按钮同排，间距 12dp、两侧边距 24dp。导入按钮在页面 Backdrop 之外采样背景，使用约 85% 黑色高光渐变衬底和 36dp 白色加号；普通主题保留原有独立悬浮导入按钮。修复正常图文层遗漏三等分权重的问题，使每组图标与标签严格以各自区域水平居中。

- 设置页的预测返回、页面入场动画、启动页开关，以及阅读器主题面板的优化排版、音量键翻页开关均接入统一组件。非液态主题仍由组件内部使用原 Material Switch，不改变既有视觉和行为。

## 阅读器底部面板液态玻璃

- 新增统一 `liquidGlassSheetSurface` 容器 Modifier，液态主题下由阅读页 Backdrop 实时采样书页，执行 vibrancy、blur、lens、高光轮廓和内外阴影；面板衬底浓度随用户设置的玻璃透明度联动。普通主题继续使用 24dp 阴影实体卡片。
- EPUB/TXT 的目录、搜索、笔记编辑、书签/笔记列表、主题和高级设置，以及 PDF 的目录和转换面板，共 8 个底部上滑面板接入统一玻璃容器。
- 高级设置按文字颜色、文字排版、页边距、页脚信息与按键、字体拆分为五个信息组。液态主题下每组叠加第二层半透明深/浅渐变色块和轻量高光边框，与外层玻璃衬底共同保证文字、滑条和开关可读；普通主题不额外绘制分组色块。
- 底部面板入场与退出改用统一弹簧：位移、轻微缩放、透明度和页面蒙版由同一进度联动，入场有克制回弹，退出同步向下收走。打开任一底部面板时 TTS 胶囊同时退出，主题面板切换到高级设置时旧层退出与新层进入同帧交叉，避免透明容器叠加停留。
- TTS 倍速菜单取消独立窗口 `DropdownMenu`，改为主胶囊上方同一页面坐标系内的玻璃浮层，修复折射背景位置偏移。倍速浮层使用普通阅读胶囊约 18% 的浅衬底，只有 TTS 主胶囊保留 85% 深衬底；展开和收起采用高度、位移、缩放与透明度联动的弹性动画。
- 文字选择操作菜单同样取消独立 `Popup` 实体背景，改为阅读页内按选区坐标绝对定位的液态玻璃胶囊；操作菜单与颜色选择器的宽度切换使用弹簧，避免跨窗口 Backdrop 坐标偏移。位于阅读页玻璃提供层之外的笔记输入和书签/笔记列表显式接收书页 Backdrop，保证实际折射来源一致。
- 根据真机可读性反馈将底部信息面板的第一层深/浅衬底在默认透明度下由约 41% 提高到约 65%；即使用户将玻璃透明度调到最高，信息面板衬底也不低于约 52%。TTS 倍速小浮层仍保持普通胶囊的浅衬底，不套用面板加深值。
- 液态玻璃下的底部信息面板改为完整四圆角悬浮容器，左右内收 `14dp`、底部悬空 `12dp`，增加独立外投影、更清晰的高光轮廓和玻璃边缘高光；普通主题仍保持原有贴底面板。
- 悬浮容器使用比用户设定值低 `10%` 的实际玻璃通透度，衬底浓度继续保留信息可读性下限；目录、搜索、笔记、主题、高级设置、PDF 目录和转换容器统一生效。
- 底部容器进入时从屏幕下方向上移动，并从约 `92% x 90%` 的缩小状态同步放大；弹簧越过落点时允许轻微放大后回到原尺寸。退出过程反向联动位移、缩放与透明度，避免先移动再变形的割裂感。
- 悬浮容器的信息蒙版在默认透明度下提高到约 70%，透明度拉满时仍保留约 58%，仅加深内容承载层，不削弱背景折射和轮廓高光。
- 根据真机动效反馈，将悬浮容器入场初始缩放由约 `92% x 90%` 加强为统一 `86%`，并降低进入和退出弹簧刚度；缩放变化更容易感知，整体移动节奏更从容，仍保留越过落点后的弹性收束。
- 修正液态玻璃开关滑块的静止位置：在原有 `20dp` 行程之外补充 `3dp` 固定起点，使开启时右边距、关闭时左边距和上下边距均为 `3dp`，白色滑块不再贴边。关闭轨道改用中性灰色玻璃衬底，拖动时再连续过渡到 Lumi 粉色。

## 书架长按菜单与页面内展开菜单

- 书架长按菜单移除独立 `Popup` 和黑色压暗层，改为书架页面内的清晰覆盖层；原书架内容继续随长按进度高斯模糊，放大的封面、资料面板和操作面板保持清晰。资料与操作面板接入书架独立 Backdrop，使用真实背景折射、内容衬底、阴影和高光轮廓。
- 长按模糊进度提升到主导航层，主页底部 Tag 栏与导入按钮和书架内容使用同一进度同步模糊，不再在菜单出现时隐藏或保持锐利；整个过程不绘制深色蒙版。
- 新增统一页面内液态玻璃菜单宿主，按锚点在根页面坐标系定位，自动判断向上或向下展开，点击空白处关闭，并使用缩放、透明度联动的弹性进出动画。主页继续阅读菜单、设置选项菜单、阅读器高级设置页脚信息菜单均接入；普通主题继续使用原 Material `DropdownMenu`。

- 调整书架 Backdrop 与高斯模糊的绘制顺序，使长按菜单采样已经模糊的书架画面，不再在模糊背景上透出清晰内容。底部 Tag 栏和导入按钮的 RenderEffect 从控件局部边界提升到整屏透明图层，消除 Tag 栏上方因模糊采样边界产生的横向断层线。

## 液态玻璃拖动条

- 重构通用 `PillSlider` 的手势状态：拖动阶段直接维护连续浮点值并逐帧跟手，不再按刻度回调业务层；松手后才按控件步长计算最近落点，通过弹簧动画从当前位置吸附，并只提交一次最终值。
- 液态主题下轨道与深色活动段均接入当前页面 Backdrop。深色活动段使用半透明黑色衬底、折射、阴影和高光轮廓，按下后横向与纵向弹性放大，拖动时保持棱镜状态；普通主题继续使用原实体胶囊轨道。
- 为字号、亮度、行高、字间距、页边距、颜色和每日目标补齐对应吸附步长。高级阅读设置的数值文本在拖动时仅更新本地状态，透明度设置在拖动时仅更新内存预览，避免高频写入 DataStore；松手吸附完成后再持久化。
- 液态主题高级设置的五类分组背景有效不透明度提高约 `5%`，让内部小色块更容易辨认，同时保留与外层容器叠加后的通透感和高光轮廓。
- 根据真机静态截图重新拆分滑条材质层：未按压时恢复实色轨道与填充，浅色模式为灰底黑色活动段，深色模式为黑底白色活动段；按住后实色活动段才交叉淡出为半透明折射棱镜，并随按压进度放大。活动段继续采用整条胶囊裁切方案，右端保持垂直切口，不再绘制成圆弧。
- 主题面板进一步拆分为书页折射玻璃底层与控件内容层，并为底层建立独立 Backdrop。面板内滑条、自动亮度按钮和“优化书籍排版”开关改为采样容器玻璃，避免直接折射书页文字导致控件内容混乱；自动亮度按钮在液态主题下使用与滑条一致的浅色黑/深色白玻璃配色。
- 阅读背景横向选项以玻璃容器完整宽度作为滚动视口，左右 `24dp` 留白改为可随内容滚动的内部间距；滑动到两端时，圆形选项只会在容器边缘被裁切，不再提前在正文内边距处截断。
- 滑条未选中的底槽恢复轻量常驻玻璃，而非完全实色：浅色模式使用低浓度中性灰折射，深色模式使用较浅的半透明黑色折射，并提高实际通透度；活动段仍保持静止实色、按住后才切换为放大的折射棱镜。
- 液态主题书架长按菜单为底部 72dp Tag 栏、悬浮间距和系统导航区预留更大的安全范围。上下两块菜单在靠近屏幕底部时整体上移，保持在模糊底栏之上，不再让底栏覆盖最后一项顶层操作。
- 高亮书籍封面按放大后的实际视觉底边计算底栏安全区，并使用独立 `coverPositionProgress` 驱动位置。长按最后一行书籍时，封面从原坐标边放大边弹性上移；退出时与缩小同步平滑回到原坐标，完成后才交还给网格原封面，消除入场和退出瞬移。

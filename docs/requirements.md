# 电子书阅读器 - 需求文档

## 版本信息
- 版本：v1.1
- 日期：2026-06-18
- 状态：已更新

## 一、功能需求

### 1.1 书籍管理
**支持格式**
- EPUB：标准电子书格式
- PDF：便携式文档格式
- TXT：纯文本格式

**导入方式**
- 从手机本地存储选择文件导入
- 支持批量导入

**书籍信息**
- 封面：从文件元数据自动提取
- 标题：从文件元数据读取
- 作者：从文件元数据读取

### 1.2 首页
**布局**
- 顶部：应用标题、今日阅读时长
- 中间：书籍网格展示（封面、标题、作者）
- 底部：标签栏（首页、统计）

**功能**
- 导入书籍按钮
- 书籍网格/列表视图切换
- 书籍搜索
- 书籍排序

### 1.3 统计页
**数据展示**
- 今日阅读时长
- 本月总阅读时长
- 日阅读目标（可设置）
- 目标完成百分比

**图表**
- 柱状图：展示每日阅读时长
- 时间范围：最近7天/30天

### 1.4 阅读界面
**全屏模式**
- 隐藏Android状态栏
- 隐藏导航栏
- 沉浸式阅读体验

**页面设计**
- 圆角矩形页面
- 舒适的阅读区域

**翻页效果（Apple Books 风格平移覆盖）**
- 拖拽翻页：手指拖拽时页面实时跟随，平整滑动无 3D 形变
- 边缘阴影：两页交界处有细腻的动态渐变阴影，制造纸张叠加的高低差错觉
- 吸附/回弹：松手后根据距离（>30%屏宽）或速度（>500px/s）决定完成翻页或回弹
- 橡皮筋效果：在章节边界继续拖拽时有阻尼反馈
- 跨章节翻页：滑出当前章节 → 加载新章节 → 从对侧滑入
- 单击翻页：左侧 30% 上一页，右侧 30% 下一页
- 单击中间 40%：显示/隐藏菜单栏

### 1.5 阅读菜单
**目录功能**
- 章节列表
- 快速跳转

**字体设置**
- 字体类型：系统字体选择
- 字体大小：可调节
- 行距：可调节
- 字间距：可调节

**主题设置**
- 背景颜色：多种预设
- 文字颜色：多种预设
- 自定义颜色

**亮度调节**
- 应用内亮度调节
- 跟随系统亮度

**书签功能**
- 添加书签
- 书签列表
- 快速跳转

**笔记和高亮**
- 文字选择
- 高亮标记
- 添加笔记
- 笔记列表

**搜索功能**
- 全文搜索
- 搜索结果高亮

**阅读进度**
- 当前章节
- 总进度百分比
- 阅读位置

## 二、非功能需求

### 2.1 性能要求
- 应用启动时间 < 2秒
- 翻页响应时间 < 100ms
- 书籍加载时间 < 3秒

### 2.2 兼容性
- 最低Android版本：8.0 (API 26)
- 目标Android版本：14 (API 34)
- 支持设备：手机和平板

### 2.3 存储
- 书籍文件本地存储
- 阅读进度本地保存
- 笔记和书签本地保存

### 2.4 用户体验
- 流畅的动画效果
- 直观的操作方式
- 舒适的视觉体验

## 三、用户故事

### 3.1 书籍管理
- 作为用户，我想从手机导入电子书，以便在应用中阅读
- 作为用户，我想看到书籍的封面、标题和作者，以便快速识别书籍
- 作为用户，我想搜索和排序书籍，以便快速找到想读的书

### 3.2 阅读体验
- 作为用户，我想全屏阅读，以便沉浸式体验
- 作为用户，我想通过滑动翻页，以便自然地阅读
- 作为用户，我想调整字体和主题，以便舒适地阅读

### 3.3 阅读统计
- 作为用户，我想看到阅读时长，以便了解阅读习惯
- 作为用户，我想设置阅读目标，以便保持阅读动力
- 作为用户，我想查看阅读统计，以便分析阅读情况

## 五、约束和限制

### 5.1 技术约束
- 使用Kotlin开发
- 使用Jetpack Compose UI框架
- 遵循MVVM架构

### 5.2 业务约束
- 第一版不支持网络功能
- 不支持云同步
- 不支持DRM保护书籍

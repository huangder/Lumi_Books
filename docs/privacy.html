隐私政策

生效日期：2026年6月29日 · 最近更新：2026年7月21日

一、引言

Lumi（以下简称"本应用"）是一款本地电子书阅读器。我们深知隐私对您的重要性，本隐私政策旨在向您说明我们如何处理您的个人信息。

核心原则：本应用坚持本地优先。只有在您主动启用并选择 MinerU 云端解析时，您选定的 PDF 才会上传至第三方服务；本应用不会在后台自动上传文件。

开源透明：本应用为开源项目，完整源代码公开可审查。您可以随时查阅源代码，验证本隐私政策所述内容的真实性。我们相信，代码透明是最好的隐私保障。

二、权限说明

本应用仅申请以下 Android 系统权限，且每项权限均有明确的用途说明：

权限用途是否必要
READ_EXTERNAL_STORAGE（Android 12 及以下）读取您设备上的电子书文件（EPUB、TXT、PDF）是
READ_MEDIA_IMAGES（Android 13 及以上）读取电子书中的封面图片资源是
INTERNET检查应用更新及条款/政策变更；在用户主动选择时连接 MinerU 云端解析否（可选功能）

本应用不申请以下权限：

  • ✕ 位置信息权限
  • ✕ 相机/麦克风权限
  • 💡 TTS 语音朗读使用 Android 系统文字转语音引擎，文本内容在设备本地处理，不采集、不录制、不上传任何音频数据。
  • ✕ 通讯录权限
  • ✕ 电话状态权限
  • ✕ 写入外部存储权限
  • ✕ 任何其他敏感权限

三、数据处理说明

3.1 本应用处理的数据

以下应用设置和阅读数据存储在您的本地设备上；主动使用 MinerU 时的文件传输另见 3.2：

数据类型说明存储方式
电子书文件您主动选择打开的 EPUB、TXT、PDF 文件本地文件
阅读进度当前阅读章节和页码位置Room/SQLite
书签记录您手动添加的书签Room/SQLite
阅读时长每次阅读的开始时间、结束时间、持续时长Room/SQLite
阅读偏好字号、主题、行距等显示设置DataStore
MinerU 设置解析模式、同意版本和时间；个人 Token 使用 Android Keystore 加密本地 DataStore/加密存储
TTS 朗读文本传递给系统 TTS 引擎的电子书文本（仅内存中处理，不持久化存储）本地内存（不存储）
最近阅读最近打开的书籍列表及时间Room/SQLite

3.2 网络使用说明

  • 仅在以下场景发起网络请求：
    • 启动时或手动触发"检查更新"——请求 GitHub Raw 上的 update_config.json
    • 打开 GitHub Releases 页面下载新版本（需用户手动点击）
    • 用户主动选择 MinerU 云端解析——向 MinerU API 申请上传地址，将所选 PDF 上传至其 OSS 存储，轮询解析进度并从 CDN 下载结果
  • 更新检查请求不携带个人信息、设备标识或使用数据
  • MinerU 请求会向第三方传输所选 PDF、随机化文件名、网络 IP、解析参数和任务标识；精准解析还会传输用户个人 Token
  • 本应用不使用分析、广告或用户追踪 SDK，也不会将 Token 或 PDF 发送给 LumiBooks 开发者

3.3 本应用不处理的数据

  • 不收集您的个人身份信息（姓名、邮箱、电话等）
  • 不收集您的设备标识信息（IMEI、Android ID、MAC 地址等）
  • 不收集或接收您的阅读内容；启用 MinerU 时，文件由设备直接发送给 MinerU
  • 不使用任何分析或追踪工具（如 Google Analytics、Firebase 等）
  • 不使用任何广告 SDK
  • 除您明确发起的 MinerU 解析外，不进行文件上传或云同步

四、数据存储与安全

4.1 存储方式

本地阅读数据存储在应用的私有沙盒目录中，遵循 Android 系统的应用隔离机制。MinerU Token 使用 Android Keystore 加密并排除系统备份。

4.2 数据安全

  • 应用仅在检查更新及您主动选择 MinerU 云端解析时联网
  • 数据存储在 Android 应用沙盒中，受系统级隔离保护
  • 本应用不收集任何可识别您身份的信息

4.3 数据保留

  • 您可以随时在应用内清除阅读历史、书签等数据
  • 卸载应用将自动删除所有应用数据
  • 电子书源文件由您自行管理，本应用不会修改或删除您的原始文件

五、第三方服务

本应用不集成分析、广告、推送、云同步或崩溃报告服务。您可以主动启用由上海人工智能创新中心运营的 OpenDataLab/MinerU 云端解析服务。MinerU 与 LumiBooks 独立运营，使用该服务受其服务协议和个人信息保护政策约束。

如不希望文件离开设备，请保持 MinerU 关闭并使用本地解析。您可随时停用 MinerU、清除个人 Token，并联系其运营方行使平台数据权利。

除此之外，本应用：

  • 无第三方分析服务
  • 无第三方广告服务
  • 无第三方推送服务
  • 无云同步服务
  • 无崩溃报告服务

六、儿童隐私

本应用不针对 13 岁以下儿童，不会故意收集儿童的个人信息。如果您是儿童的监护人，发现儿童可能向本应用提供了个人信息，请联系我们，我们将及时删除相关信息。

七、隐私政策的变更

如果我们决定更新本隐私政策，我们将在本页面发布更新后的版本，并更新"最近更新"日期。建议您定期查看本隐私政策以了解最新信息。重大变更将以应用内通知的方式告知您。

八、联系我们

如果您对本隐私政策有任何疑问、意见或建议，请通过以下方式联系我们：

  • 应用内反馈：通过应用设置页面提交反馈
  • 电子邮件：huangder0104@126.com
  • 开源社区：在代码仓库提交 Issue 反馈问题或建议

隐私验证：作为开源项目，我们鼓励您通过审查源代码来验证本隐私政策的真实性。所有数据处理逻辑均可在代码中查阅。

九、适用范围

本隐私政策仅适用于 Lumi 应用（com.huangder.lumibooks）在 Android 平台上的功能。不适用于通过其他方式获取您信息的第三方服务。
